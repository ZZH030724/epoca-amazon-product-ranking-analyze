# Amazon Product Trend Analysis

## Overview
Analyzed 10,000+ Amazon product rankings to identify high-growth product opportunities.

## Data Pipeline
Raw Data → Cleaning → Feature Engineering → Trend Detection → Insights

## Key Techniques
- Pandas / NumPy
- Feature Engineering
- Zipf’s Law
- Z-score normalization

## Key Findings
- Identified breakout product: Anti-fatigue kitchen mat
- Sustained 15–30 rank increase over 4–6 weeks

## Business Impact
- Supports product selection decisions
- Identifies emerging trends early
